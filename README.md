## What is Za Inspector?

Za Inspector is a Chrome extension designed for Zoho CRM users who need to quickly inspect record data, run COQL queries, and perform bulk import/export operations. Whether you're a CRM administrator or power user, Za Inspector gives you the tools to work more efficiently with your Zoho CRM data.

---

## Features

### Record Inspection
Instantly view all field data for any Zoho CRM record including Field Name, API names, data types, and Export record info in CSV/XLSX/JSON.

### Export Data
Write and execute COQL queries with a professional editor featuring syntax highlighting, query templates, save queies, history tracking, Export result in CSV/XLSX/JSON.

### Data Import
Import data with a guided 4-step wizard supporting CSV, JSON, and XLSX files with intelligent field mapping.

### Theme Support
Choose between Light, Dark, or System theme to match your preference.

### Admin Permissions
Administrators can control feature access per Zoho profile with granular permissions.

### Multi-Datacenter Support
Works with all Zoho datacenters: US, EU, IN, AU, JP, CN, SA, and CA.

### Multi-API versioning Support
Works with all Zoho CRM APIs Versions: V2, V2.1, V3, V4, V5, V6, V7, V8.

## Usage

### Getting Started

1. **Login**: Click the Za Inspector icon and click "Login to Zoho CRM"
2. **Authorize**: Grant the required permissions when prompted
3. **Navigate**: Go to any Zoho CRM record page
4. **Inspect**: Click the extension to view record details

### Viewing Record Data

When you're on a Zoho CRM record page:
- Click the extension icon to see a summary of the record
- Click **View Full Data** to see all fields with their API names and values
- Export the data in your preferred format

### Running COQL Queries

1. Click **Export Data** from the main popup
2. Write your COQL query or use a template
3. Click **Run Query** to execute
4. View results in the table and export as needed
5. Export the data in your preferred format

### Importing Data

1. Click **Import Data** from the main popup
2. Choose the target module and operation (Insert/Update/Delete)
3. Upload your file or paste data
4. Map your columns to Zoho CRM Module fields
5. Preview and confirm the import

### Settings

- View your current datacenter
- Change the API version
- Switch between Light/Dark/System themes

## Permissions

Za Inspector requires access to your Zoho CRM account to:
- Read and write CRM records
- Access module and field metadata
- Execute COQL queries
- Read organization and user information

You can revoke access at any time from your Zoho account settings.

---

## Privacy

- Za Inspector only accesses your Zoho CRM data when you're actively using the extension
- Your data stays between you and Zoho
- OAuth tokens are stored securely in your browser
- No analytics or tracking
